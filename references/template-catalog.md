# Template Catalog

This skill includes four HTML template assets. Choose one before full rendering and confirm with the user through a demo.

## Template 0: final-course
Asset: `assets/templates/template-0-final-course.html`

Use for:
- formal internal training
- practical AI enablement courses
- chaptered decks with many pages
- decks needing top navigation, progress, anchors, and one-screen sections

Style:
- white background
- blue and teal accents
- clean cards
- 16:9 full-page sections
- refined but restrained interactions

Default choice when the user does not specify a style.

## Template 1: blue-pink
Asset: `assets/templates/template-1-blue-pink.html`

Use for:
- friendly modern sharing
- introductory AI or workplace productivity topics
- soft, approachable training

Style:
- blue-pink gradient
- right-side pagination dots
- soft card shadows
- warm and approachable visual tone

Watch for:
- avoid overusing hover scale
- keep font large enough for projection

## Template 2: blue-white
Asset: `assets/templates/template-2-blue-white.html`

Use for:
- technology or tool training
- clean professional demos
- Codex, automation, data, or workflow decks

Style:
- blue-white gradient
- strong technology feel
- card grids and scenario cards
- vertical pagination dots

Watch for:
- dense scenario pages may need compact layout
- code blocks should wrap and remain readable

## Template 3: white-pink
Asset: `assets/templates/template-3-white-pink.html`

Use for:
- warm internal training
- HR, admin, general workplace adoption sessions
- content that should feel less technical

Style:
- white-pink-yellow gradient
- right sidebar navigation
- reveal animations
- rounded translucent cards

Watch for:
- right sidebar can reduce content width
- control card height carefully on dense pages

## Choosing a template
If the user gives no preference:
1. Use template 0 for formal course decks.
2. Use template 2 for technical tool or Codex training.
3. Use template 1 for light sharing and friendly productivity themes.
4. Use template 3 for warm, people-oriented internal enablement.
