# User Prompt Template

When the user wants to start a new deck, ask for or infer:

```text
主题：
受众：
培训/展示目标：
希望页数：
已有材料：
是否需要章节页：
喜欢的模板：final-course / blue-pink / blue-white / white-pink / 不确定
输出形式：逐页 HTML / 单文件 HTML / 先出 demo
限制：比如必须 16:9、一屏展示、字体大、不能裁切、动效轻量
```

If the user only gives a rough idea, proceed with a reasonable assumption and state it in the planning draft.
