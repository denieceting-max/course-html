Codex：你的私人编程助理

Codex是什么？

OpenAI推出的代码生成引擎，GPT-4已深度集成。你把需求用中文说出来，它输出可直接运行的代码。

怎么用？

打开ChatGPT（GPT-4），直接说写一段Python代码，复制代码到电脑，运行，完成。

它擅长什么？

文件批量重命名、格式转换、数据提取整理汇总、Word/Excel/PDF自动化处理、邮件批量生成、网页信息抓取。

一句话总结

Codex就是你的代码翻译官，中文进去，代码出来。

Codex实战①：批量合同整理

场景：每月50份PDF合同，需按甲方名称+签订日期重命名归档

传统做法

逐个右键重命名，2小时起步，眼花手酸

Codex做法（5分钟）

1. 打开ChatGPT输入需求
2. 复制代码运行
3. 50个文件瞬间整齐排列

写一段Python代码，读取D:\合同\文件夹下所有PDF文件，提取文件名中的甲方名称和签订日期（格式如"XXX公司_2025.01.01"），将文件重命名为"甲方名称_签订日期.pdf"

你只需要：会复制、会粘贴、会双击

Codex实战②：客户信息提取

场景：100份Word合同，需提取所有客户名称、联系方式到Excel

传统做法

逐个打开、复制、粘贴，一上午没了

Codex做法（3分钟）

1. 在ChatGPT中输入需求
2. 运行代码，Excel自动生成

写一段Python代码，读取D:\合同\文件夹下所有.docx文件，从每个文件中提取"甲方全称"和"联系电话"两个字段，汇总到一个Excel表格中，列名为"客户名称"和"联系方式"

举一反三：同样的方法，可以批量提取报价单、简历、问卷中的任何信息

Codex实战③：邮件批量生成

场景：通知50位供应商更新资质文件，每个邮件需带对方公司名和截止日期

传统做法

逐个复制粘贴修改，1小时+

Codex做法（10分钟）

1. 准备Excel：列名供应商名、联系人、邮箱、截止日期
2. 在ChatGPT中告诉Codex
3. 运行，批量生成邮件

读取这个Excel，为每一行生成一封邮件，主题为"关于更新资质文件的通知"，正文包含"尊敬的[供应商名]您好，请于[截止日期]前完成资质更新"，保存为.eml草稿文件

50封邮件，10分钟搞定，零出错

Codex实战④：数据报表一键生成

场景：行政季度满意度调查，200份Excel结果需统计平均分、各部门对比

传统做法

数小时拉公式、做透视表、画图表

Codex做法（5分钟）

1. 在ChatGPT中输入需求
2. 运行，图表和统计结果一次性输出

写一段Python代码，读取D:\调查结果.xlsx中第3列"满意度评分"，计算整体平均分、各部门平均分，生成一个柱状图（蓝色，标题"各部门满意度对比"），保存为png

即使完全不懂Excel函数，跟着AI提示就能做出专业报表

Codex写好提示词的秘诀

1. 明确角色

你是一个Python编程专家……

2. 给出上下文

我有一个Excel文件，放在D盘，列名是……

3. 指定输出格式

请输出完整的Python代码，加上中文注释

4. 提供例子

如果公司名是"阿里巴巴"，输出格式应为"阿里_合同.pdf"

辅助工具① Cursor：像聊天一样写代码

Cursor是什么？

AI原生代码编辑器（基于VS Code），安装即用。按Ctrl+K打开对话框，用中文描述需求。AI自动生成代码，并解释每一行。

行政适用场景

想要一个可视化界面（比如加班申请网页）、需要反复修改调试的脚本、边写边看效果的需求。

和Codex的关系：Codex生成代码片段，Cursor帮你管理和运行整个项目

辅助工具② Cloud Code：搭建专属小应用

Cloud Code是什么？

Google Cloud的AI开发插件。直接开发并部署到云端，生成可访问的网址。

行政适用场景

会议室预约页面、物品领用登记系统、内部FAQ问答机器人、问卷收集与自动统计。

实战案例：搭一个"行政问答机器人"

  • 1. 收集常见问题（请假流程、报销规范）和标准答案
  • 2. 在Cloud Code中用AI生成问答网页
  • 3. 部署后链接发到群里，同事自助查询
  • 4. 24小时在线，行政从此不用当"复读机"

安全与边界

数据隐私

涉及员工信息、合同、财务数据，优先使用本地运行（本地Python+Cursor），不要将敏感数据上传到公有云AI平台。

代码审查

AI生成的代码执行前最好扫一眼，确保不会误删文件、误发邮件。

合规提醒

批量发送邮件前，确认符合公司信息安全规定。搭建内部网页时，设置访问密码，不对外开放。

权限控制

自动化脚本仅在个人工作目录运行，涉及共享文件夹操作前，先做备份。

完整案例：行政周报自动生成器

需求：每周五收集各部门周报，自动合并为总周报

实现步骤：

  • 1. 各部门周报放共享文件夹（统一命名规则）
  • 2. 用Codex生成脚本：读取所有文档，提取"本周重点""下周计划"段落
  • 3. 合并为一个Word文档，自动添加封面和目录
  • 4. 可选：自动发送邮件给经理
